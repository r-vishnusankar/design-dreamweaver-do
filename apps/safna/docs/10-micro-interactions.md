# Micro-interactions

| Interaction | Feedback | Location |
|-------------|----------|----------|
| Begin the Experience | Fade overlay exit | Sound gate |
| Scroll indicator | Gentle bounce | Hero |
| Premium card hover | Border gold tint | Details, Family |
| Gallery image hover | Scale 1.03 + caption | Gallery |
| Gallery click | Blur backdrop + scale in | Lightbox |
| Blessing submit | Flower bloom animation | Blessings |
| Mute toggle | Icon swap Volume2/VolumeX | Fixed bottom-right |
| Long-press seal | Petal burst + hidden dua | Thank You |
| Cursor move (desktop) | Jasmine petal trail | Global |
| Countdown tick | Number fade-in | Countdown |
| Night mode | CSS variable shift | After 6 PM |

## Haptic Considerations (Mobile)
- Long-press seal works on touch
- No vibration (respectful for wedding context)

## Sound Design
- Ambient piano loop at 30% volume
- User must opt in via "Begin the Experience"
- Mute persists during session

## Easter Eggs
1. **Long-press wedding seal** → Quranic dua bloom
2. **Night mode** → Activates after sunset (6 PM–6 AM)
3. **Hidden geometric animation** → Subtle background in Thank You / Hero
