# Design System

## Color Tokens

| Token | Hex | Usage |
|-------|-----|-------|
| `ivory` | `#FAF8F5` | Primary background |
| `champagne` | `#F5EDE3` | Section alternates |
| `warm-white` | `#FFFEF9` | Cards |
| `sage` | `#B8C4B0` | Accents, petals |
| `sage-muted` | `#D4DDD0` | Subtle backgrounds |
| `gold` | `#C4A962` | Primary accent |
| `gold-muted` | `#D4C4A0` | Gradients |
| `sand` | `#E8DFD0` | Borders |
| `kerala-green` | `#7A8B72` | Nature accents |
| `text-primary` | `#2C2825` | Headings |
| `text-secondary` | `#6B6560` | Body |
| `text-muted` | `#9A948C` | Labels |

## Typography Scale

| Role | Font | Weight | Usage |
|------|------|--------|-------|
| Display | Great Vibes | 400 | Names, quotes |
| Heading | Cormorant Garamond | 300–600 | Titles, family names |
| Body | Inter | 300–500 | Paragraphs, labels |

## Spacing

- Section padding: `py-24` (96px)
- Card padding: `p-8 md:p-10`
- Label tracking: `0.3em` uppercase
- Max content width: `max-w-4xl` for cards, `max-w-2xl` for prose

## Components

- `.chapter-container` — Full viewport section wrapper
- `.premium-card` — Frosted card with sand border
- `.gold-divider` — 64px centered line
- `.chapter-label` — "Chapter 01" style label

## Complement to Groom Site

Groom site uses similar serif elegance and gold accents. Bride site adds **sage/ivory warmth** and **editorial whitespace** — softer, more intimate, while sharing the same factual content structure.
